## Install

```bash
# 克隆仓库
git clone https://github.com/your-username/novel-reviewer.git
cd novel-reviewer

# 安装依赖并注册全局启动命令
bun install
bun link
```

> 没有 Bun？前往 [bun.sh](https://bun.sh) 安装：`curl -fsSL https://bun.sh/install | bash`

## Start

```bash
# 打开书架
cr
```

## Shortcuts

| 按键 | 功能 |
|------|------|
| `j` / `↓` | 下一段 |
| `k` / `↑` | 上一段 |
| `d` / `u` | 半屏翻页 |
| `Space` | 整屏翻页 |
| `n` / `N` | 下/上一章 |
| `g` / `G` | 跳到顶部/底部 |
| `:toc` | 打开目录 |
| `/` | 搜索章节 |
| `q` | 退出 |
